# mealmingle_frontend

to run mealmingle frontend
(runs on port 9000)

``` 
git clone https://github.com/ramjayakumar21/mealmingle_frontend
cd mealmingle_frontend
npm ci
npm run start
```

to run mealmingle backend
(runs on port 9001)

``` 
git clone https://github.com/ramjayakumar21/mealmingle_backend
cd mealmingle_frontend
npm ci
npm run devstart
```
